Direct installation

Procademy developed with Laravel Php framework, therefore, you should install Laravel fundamentals on the server according to this list:

  • PHP >= 7.1.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Using Linux based OS is preferred specially CentOS distribution.

Update the Php version when new updates released.

If you are using server managers like “Easy Apache”, you can install all extensions and dependencies.

Proacademy exclusive dependencies:

FFMpeg: Install the latest FFmpeg version from the official website. If you didn’t find your OS compatible version, you can basic as a binary distribution.

Proacademy direct installation:

You can install Proacademy directly without server manager applications like CPanel or Direct Admin. You can also run the server without using Apache or Nginx by NodeJs.

  • First, install NodeJs
  • Install PHP CLI (Php Command line)
  • Install the composer. You can use composer to install or update dependencies.
  • Unzip Proacademy package on the server.
  • Run command line
  • You can run the server with PHP artisan serve code when code ran website will be available through 3000 Laravel port.
  • If you want to run the server through 80 port, you can run the server with this code:

php artisan serve --port=80
